# 🎮 Horrimania - Game Download Website (Frontend Project)

## 📌 About the Project
Horrimania is a simple frontend website project built using HTML and CSS.  
The concept of this project is a game download website where users can explore horror games.

This project focuses only on frontend design and layout, with no backend functionality.

---

## 🛠️ Technologies Used
- HTML 
- CSS
- Visual Studio Code

---

## 🎯 Features
- Clean and simple UI design  
- Horror game themed layout  
- Game download website interface design  
- Multi-page structure using HTML  
- Styling implemented with CSS  

---

## 🎮 Project Idea
This website showcases horror games such as:
- Resident Evil series  
- Other popular horror games  

It is presented only as a frontend design concept (no real downloads).

---

## ⚠️ Note
This is a **frontend practice project only**.  
It does not include any backend system or real game download functionality.
